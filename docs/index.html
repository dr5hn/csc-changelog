About This Project

This changelog system tracks every modification to the Countries States Cities Database by analyzing git commit history.

All changes are automatically extracted from the world.sql file and organized by country. Recent changes (last 24 months) are readily available, with older changes archived by year.

Data Retention

  • Active: Last 24 months in main changelog files
  • Archived: Older changes organized by year
  • Format: Minified JSON with standard field names
